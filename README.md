# extitution.io

the other half of the barbell. an independent nonprofit growing open, networked, community-shaped coordination at ethereum's app layer.

live at [extitution.io](https://extitution.io)

## what's here

a single-page static site — everything lives in `index.html`. deploys to vercel via `deploy.sh`.

## contributing

1. fork the repo (or branch, if you have write access)
2. edit `index.html` — keep the tone and style consistent with what's there
3. preview locally by opening `index.html` in a browser (no build step needed)
4. open a pull request against `master` with a short description of the change

small fixes (typos, broken links) are welcome as direct PRs. for bigger changes — new sections, restructuring, messaging — come talk it through in the telegram first.

## deploying

maintainers only:

```sh
./deploy.sh "commit message"
```

commits, pushes to github, and ships to vercel production.
